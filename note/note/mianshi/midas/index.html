以下是你简历中的机器学习相关词汇的中英双语解释：

1. Deep Learning (深度学习)

Deep learning is a subset of machine learning that uses neural networks with many layers to model and understand complex patterns in data. 深度学习是机器学习的一个子集，它使用具有多层结构的神经网络来建模并理解数据中的复杂模式。

2. Convolutional Neural Network (CNN) (卷积神经网络)

CNN is a type of deep neural network used primarily for analyzing visual data, such as images. It is designed to recognize patterns by applying convolutional layers that filter the input. 卷积神经网络是一种主要用于分析视觉数据（如图像）的深度神经网络。它通过卷积层过滤输入数据来识别模式。

3. Transformer (变换器模型)

The Transformer model is a neural network architecture that uses self-attention mechanisms to capture dependencies in sequential data, commonly used in natural language processing and, more recently, in vision tasks. 变换器模型是一种神经网络架构，使用自注意力机制捕捉序列数据中的依赖关系，常用于自然语言处理，最近也被用于视觉任务。

4. ResNet (残差网络)

ResNet, or Residual Networks, are deep neural networks designed to handle the degradation problem in very deep models by introducing skip connections. 残差网络是通过引入跳跃连接来处理非常深的模型中退化问题的深度神经网络。

5. LSTM (长短期记忆网络)

LSTM is a type of recurrent neural network (RNN) that can learn long-term dependencies in sequential data by using gates to regulate the flow of information. 长短期记忆网络是一种递归神经网络（RNN），通过使用门机制来调节信息流动，从而学习序列数据中的长期依赖关系。

6. CTC (Connectionist Temporal Classification) (连接时序分类)

CTC is a loss function used in sequence-to-sequence models, especially for speech and handwriting recognition, to allow the model to predict unaligned sequences of varying lengths. 连接时序分类是一种用于序列到序列模型（特别是语音和手写识别）的损失函数，它允许模型预测不同长度的不对齐序列。

7. Attention Mechanism (注意力机制)
