双边 Z 变换概念总结

数字信号处理涉及时域和频域两个方面。在频域中，我们可以使用多种变换方法，每种方法都有其特定的应用。双边 Z 变换是其中一种重要的工具，它被定义为：

X(z)=∑n=−∞∞x(n)z−nX(z) = \sum_{n=-\infty}^{\infty} x(n)z^{-n}X(z)=n=−∞∑∞​x(n)z−n

这里，$z$ 是复数变量，因此 $X(z)$ 也是复数函数。

收敛域（ROC）

双边 Z 变换的收敛域是使得 $X(z)$ 的模有限的 $z$ 值的集合。由于求和过程中涉及到 $z^{-n}$，因此收敛域与 $|z|$ 有关，通常是以原点为中心的圆盘区域。

有理函数形式的 Z 变换

我们关注的 Z 变换通常是有理函数：

X(z)=P(z)R(z)X(z) = \frac{P(z)}{R(z)}X(z)=R(z)P(z)​

其中，$P(z)$ 和 $R(z)$ 是没有公因子的多项式。$P(z)$ 的根称为零点，$R(z)$ 的根称为极点。极点不能位于收敛域内。

收敛域与序列的关系

同一个 $X(z)$ 在不同的收敛域下可能对应不同的时域序列。例如，对于右边序列 $x(n) = a^n u(n)$，其 Z 变换的极点在 $z = a$，收敛域为 $|z| > |a|$。对于左边序列 $x(n) = -a^{-n} u(-n-1)$，其 Z 变换相同，但收敛域为 $|z| < |a|$。

部分分式展开与反变换

当 $X(z)$ 是有理函数且分子次数小于分母次数时，可以使用部分分式展开求逆变换：

X(z)=∑k=1NAkz−dkX(z) = \sum_{k=1}^{N} \frac{A_k}{z - d_k}X(z)=k=1∑N​z−dk​Ak​​

其中，$A_k$ 是常数，可以通过极限计算得到：

Ak=lim⁡z→dk(z−dk)X(z)A_k = \lim_{z \to d_k} (z - d_k) X(z)Ak​=z→dk​lim​(z−dk​)X(z)

Z 变换的性质

  • 移位性质：若 $y(n) = x(n - n_0)$，则 $Y(z) = z^{-n_0} X(z)$。
  • 卷积性质：若 $Y(z) = X_1(z) X_2(z)$，则对应时域卷积 $y(n) = x_1(n) * x_2(n)$。

有限长度序列的 Z 变换

对于有限长度的序列，Z 变换是一个多项式，其系数就是序列的元素。

LTI 系统的分析

对于线性时不变（LTI）系统，其冲激响应的 Z 变换 $H(z)$ 称为传递函数。输入输出关系在频域中表示为 $Y(z) = X(z) H(z)$。

  • 因果性：系统的收敛域在所有极点的外部。
  • 稳定性：系统的收敛域包含单位圆。

要同时满足因果性和稳定性，所有极点必须在单位圆内。

逆系统

两个系统互为逆系统，如果它们的冲激响应卷积为单位冲激函数 $\delta(n)$，即 $h_1(n) * h_2(n) = \delta(n)$。在频域中，这对应于 $H_1(z) H_2(z) = 1$。
